# Minesweeper Game

A modern, high-performance Minesweeper game built with React and TypeScript.

## Features

- **WJH Aesthetic**: Clean, modern UI with soft colors and dark mode support.
- **Three Difficulties**: Easy, Medium, and Hard.
- **Responsive Logic**: Fast cell reveal algorithm (DFS/Stack-based).
- **First-Click Protection**: You'll never hit a mine on your first move.
- **Flagging**: Right-click to flag potential mines.

## Development

1. `npm install`
2. `npm start`
